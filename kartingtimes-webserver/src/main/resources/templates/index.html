Microservices - Web Server

Overview

  • Simple application for viewing laptimes for racing.
  • All the all microservices use REST for communication.

The Demo

  • Eureka Dashboard: http://localhost:1111
  • Eurerka registered applications: http://localhost:1111/eureka/apps/
  • Drivers server: http://localhost:2222
  • Tracks server: http://localhost:3333
  • Laptimes server: http://localhost:4444
  • All drivers: /drivers
  • All tracks: /tracks
